# Aegis (AEG) - Base Layer 2 Security Token

Professional ERC-20 implementation on Base Mainnet.

## Quick Start
1. Install dependencies: `npm install`
2. Configure `.env` with your private key and BaseScan API key.
3. Deploy: `npx hardhat run scripts/deploy.js --network base-mainnet`

## Features
- Standard ERC-20 with 1,000,000,000 initial supply.
- Owner-only minting capability.
- Public token burning functionality.
- Emergency pause mechanism for security.

## Frontend
The dashboard is located in the `frontend/` folder. Open `index.html` to start the local interface.
