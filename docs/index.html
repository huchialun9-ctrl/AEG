Aegis (AEG) 專案簡介

Aegis (AEG) 是一個在 Base 主網 (Eth Layer 2) 上發行的工業級 ERC-20 代幣資產。本專案旨在提供高安全性、可控性且具備通縮機制的代幣解決方案。

本文件提供了開發者在整合或部署 AEG 代幣時所需的所有技術細節。

快速啟動

您可以透過 Hardhat 或 Truffle 快速初始化開發環境。

1. 安裝依賴

npm install --save-dev hardhat @nomicfoundation/hardhat-toolbox

2. 配置環境變數

請在專案根目錄建立 .env 檔案：

PRIVATE_KEY=您的錢包金鑰
BASESCAN_API_KEY=您的 BaseScan API 金鑰

智能合約架構

合約繼承自 OpenZeppelin 的標準庫，確保了最高等級的安全合規性。

  • ERC20: 標準代幣邏輯
  • Ownable: 權限控管邏輯
  • Pausable: 緊急中止機制
  • Burnable: 通縮銷毀機制
import "@openzeppelin/contracts/token/ERC20/ERC20.sol";
import "@openzeppelin/contracts/access/Ownable.sol";

contract Aegis is ERC20, Ownable {
    // 鑄造 10 億枚
    constructor() ERC20("Aegis", "AEG") {
        _mint(msg.sender, 1000000000 * 10**18);
    }
}

API 參考 / 核心函數

balanceOf(address)

GET 回傳指定地址的代幣餘額。

mint(address, uint256)

OWNER ONLY 由合約擁有者呼叫，用於增發代幣。

burn(uint256)

PUBLIC 任何持有人皆可永久銷毀自己名下的代幣。

權限與安全 (Security)

Aegis 幣內建 Pausable 機制。當偵測到鏈上異常或合約漏洞時，擁有者可以呼叫 pause() 函數暫停所有轉帳活動。

function pause() public onlyOwner { _pause(); }
